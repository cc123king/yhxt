TD-SCDMA

TD-SCDMA是英文Time Division-Synchronous Code Division Multiple Access（时分同步码分多址） 的简称，中国提出的第三代移动通信标准(简称3G)，也是ITU批准的三个3G标准中的一个，以我国知识产权为主的、被国际上广泛接受和认可的无线通信国际标准。是我国电信史上重要的里程碑。(相对于另两个主要3G标准CDMA2000和WCDMA，它的起步较晚，技术不够成熟。）虽然经过了这几年的发展，但是依然有很多可优化的地方。
